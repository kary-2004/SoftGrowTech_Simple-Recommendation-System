# SoftGrowTech_Simple-Recommendation-System
🎬🛍️ Smart Recommendation System






A simple AI-based Recommendation System built with Python and Streamlit that suggests movies 🎬 and products 🛍️ based on user preferences.



📌 Features
🔍 Searchable dropdown (no typing errors)
🎯 Filter by category (Movie / Product)
🧠 Smart recommendations based on type
📊 Interactive and clean UI
⚡ Fast and beginner-friendly
🧠 How It Works
User selects an item from dropdown
System identifies its type/category
Recommends similar items using content-based filtering
🛠️ Tech Stack
Python 🐍
Streamlit 🌐
Pandas 📊
📂 Project Structure
Smart-Recommendation-System/
│── app.py
│── README.md
⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/smart-recommendation-system.git
cd smart-recommendation-system

Install dependencies:

pip install streamlit pandas
▶️ Run the App
streamlit run app.py

Open in browser:

http://localhost:8501
